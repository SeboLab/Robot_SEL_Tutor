# Generating .txt Transcripts from .json Logs

This script finds all .json logs and converts them into .txt files in the format:

```
Misty: {Misty's speech}
USER: {User's speech}
...
```

## Instructions
1. Put .json logs from lessons that you want the transcripts for in this repository.
2. Navigate to this directory in your terminal.
3. Run `python3 transcripts.py` in the terminal.
